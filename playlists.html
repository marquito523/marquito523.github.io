About Playlist

Functionalities

ExFrame Playlists is a new system implemented within ExFrame functionalities that will allow you to save preset song queues, just like a normal queue. This system is currently only in TAS(Temporary Available State).
After a few fixes and ajustments, this feature (4 commands) will be set as a NEC (None End Command).
